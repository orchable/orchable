# 🤝 Community Hub: Sharing & Discovery

The **Community Hub** is the social core of Orchable, allowing users to discover, share, and remix AI orchestration assets.

---

## 1. Sharable Assets

- **Prompt Templates**: Individual system instructions.
- **View Components**: Custom TSX components for result visualization.
- **Orchestration Configs**: Full multi-stage pipeline bundles.
- **AI Presets**: Curated model settings for specific tasks.

---

## 2. Key Features

### Asset Discovery
Browse the hub using categories, tags, and popularity filters. Preview assets before importing them into your workspace.

### One-Click Import
The **"Use This"** button instantly clones any hub asset into your local library, complete with attribution.

### Remixing & Attribution
Every asset maintains a permanent lineage. If you modify a public asset and re-publish it, it includes a "Remixed from [Original]" badge.

---

## 3. Moderation & Safety

Orchable Hub uses a **report-based moderation** model:
1. Any user can report an asset for spam, malice, or inappropriate content.
2. Assets with ≥ 5 reports are automatically hidden from the public hub pending admin review.
3. **Sandbox Security**: All custom components run in a restricted iframe sandbox to prevent malicious code execution.

---

## 4. Access Control

| Action | Anonymous | Registered |
|---|:---:|:---:|

| Import (Use This) | ✅ | ✅ |
| Remix | ✅ | ✅ |
