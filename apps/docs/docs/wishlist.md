# 💡 Wishlist — Proposed Features

A list of proposed features and improvements, prioritized by impact and complexity.

---

## ✅ Recent Successes

- **Visual Designer (ReactFlow)**
- **Custom TSX Sandbox**
- **Authentication & RLS**
- **Asset Library (Prompts & Components)**
- **Cost Calculator**

---

## 🔴 High Priority

### 1. Unified Statistics Dashboard
Transform the home page into a real-time dashboard:
- Throughput charts (tasks/hour)
- API Key Pool health status
- Error frequency analysis

### 2. UI-based API Key Management
An **API Key Pool** tab in Settings to add, delete, and reset keys without SQL.

### 3. Prompt Version History
Rollback UI for `prompt_templates` to restore previous versions directly from the Asset Library.

---

## 🟡 Medium Priority

### 4. Dry Run / Preview Mode
Test-run a single task with mock data to view the rendered prompt and AI results instantly.

### 5. Webhook Notifications
Invoke custom webhooks when a batch is `completed` or `failed`.

---

## 🟢 Low Priority

### 6. Multi-Model Support
Support for OpenAI, Claude, and Llama via an abstraction layer.

### 7. Component Marketplace
Share Custom View Components publicly for others to import.
